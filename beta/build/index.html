Welcome to the Beta Build

Link to a steam game.

Get notified by email when your wishlisted items get released or are on sale. Remove from your wishlist

Triggers:

Wishlist Button - Towards the bottom left, below the trailer of the game

  • Hover On
  • Hover Off
  • Click
  • Click Again

Dropdown Arrow

  • Hover On
  • Hover Off

Dropdown Links

  • Hover On
  • Hover Off

Rules:

Wishlist Button

  • Hover On:
    • Cursor changes to pointer
    • Color changes to a gradient blue (-60deg, #417a9b 5%, #67c1f5 95%)
  • Hover Off:
    • Cursor changes back to default
    • Color changes back to original
  • Click:
    • Update the user's wishlist with the newly added game
    • Button shrinks to show a button with a checkbox with "On Wishlist"
    • Adds a dropdown arrow to the side of the button
    • Color changes to a gradient blue (-60deg, #417a9b 5%, #67c1f5 95%)
  • Click Again:
    • Update the user's wishlist and remove the game
    • Button grows back to default
    • Color changes back to default

Dropdown Arrow

  • Hover On:
    • Arrow turns white
    • Rotates 180 degrees
    • Background changes to a gradient blue (120deg, #5c9ac3 5%, #8ed3f7 95%)
  • Hover Off:
    • Arrow returns to original color
    • Rotates back to 0 degrees
    • Background changes back to original

Dropdown Links

  • Hover On:
    • Color fades to #2D3F49 over 0.2s, also leaves room for a slight border
  • Hover Off:
    • Color fades back to original over 0.2s
