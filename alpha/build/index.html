Welcome to the Alpha Build

Any link to a steam game would suffice so I pulled the first one I saw.

Get notified by email when your wishlisted items get released or are on sale.
Remove from your wishlist
Item added to your wishlist!

Triggers:

Wishlist Button - Towards the bottom left, below the trailer of the game

  • Hover On
  • Hover Off
  • Click

Rules:

Wishlist Button

  • Hover On:
    • Cursor changes to pointer
    • Color changes to a gradient blue
  • Hover Off
    • Cursor changes back to default
    • Color changes back to original
  • Click
    • Update the user's wishlist with the newly added game
    • Button changes to show a button with a checkbox with "On Wishlist"
    • Changes to have two dropdown options on the side of the button
      • Manage your wishlist
      • Remove from your wishlist
  • Click Again
    • Update the user's wishlist and remove the game
    • Button changes back to default

Feedback:

Wishlist Button

  • Hover On:

    • Button changes to show a button with a checkbox with "On Wishlist"

Loops:

  • None

Modes:

Wishlist Button

  • Hover On:
    • Text box appears above wishlist button with "Get notified by email when your wishlisted items get released or are on sale.""
  • Hover Off:
    • Text box disappears
  • Click:
    • Text box above the wishlist button changes to show different text with "Remove from your wishlist"
    • Another text box appears below the wishlist button with "Item added to your wishlist!"
  • Click Again:
    • Text box above wishlist button changes back to original text
    • Text box below the wishlist button disappears
